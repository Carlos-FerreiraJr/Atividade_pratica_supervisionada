# utlização

utilize os comandos a seguir para obter o programa pronto para rodar disponível  no repositório abaixo.

| repositório |  |
| ------ | ------ |
|somente executar | [clique aqui] |

 
para windows 
```sh
git clone https://github.com/Carlos-FerreiraJr/aps_
cd aps_ 
executar aps
```

para linux. 
```sh
$ git clone https://github.com/Carlos-FerreiraJr/aps_
$ cd aps_
$ java -jar "Aps.jar"
```
